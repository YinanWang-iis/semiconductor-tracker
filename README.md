# 半导体与存储产业链追踪网页

这是一个可直接打开的静态网页原型，用来帮助家人按产业链节点追踪半导体和存储行业变化。

## 怎么打开

双击 `index.html`，或在浏览器里打开：

```text
G:\semiconductor_tracker\index.html
```

## 当前功能

- 按产业链拆分：上游材料与设备、芯片设计、晶圆制造、存储制造、先进封装、光模块与服务器、下游需求。
- 按产品节点追踪：AI GPU、HBM、CoWoS、光模块、NAND、DRAM、先进制程、材料、设备等。
- 每个节点统一看三类变化：定价、库存、产能。
- 支持按地区、环节、指标和关键词筛选。
- 支持“父亲摘要模式”，减少干扰信息。
- 支持导出当前筛选结果为 CSV。

## 后续需要准备的数据

如果要把它从“网页原型”升级成真正的追踪工具，建议先准备一张表，每行一个产品节点：

```text
日期, 产业环节, 产品/工序, 公司, 地区, 定价变化, 库存变化, 产能变化, 来源链接, 可信度, 备注
```

后续可以把这张表做成 CSV/Excel，再让网页自动读取；也可以接入 RSS、公司公告、行业研究机构或新闻 API。

## 重要说明

当前内置内容是追踪框架和示例观察项，不是实时新闻或实时行情。页面里写了整理日期，避免把静态信息误认为最新数据。
